# 03_measurement_library_1q.R

## Purpose
Construct one-qubit measurement libraries (L1/L2/L3).

## Main functionality
- L1: Pauli-axis binary PVM settings.
- L2: extended axis set for richer 1q coverage.
- L3: random fixed-axis library with deterministic seed support.
- Produces setting labels, `(a,b)` index maps, and projector lists for multinomial sampling.

# 08_verify_caseI_caseII_outputs.R

## Purpose
Structural and summary verification for outputs generated by `07`.

## Checks
1. Relative-gap summaries for the asymptotic formula
   \( \mathbb{E}[D(\hat{\rho},\rho)^2] \approx \frac{1}{n}\mathrm{tr}(G_D(\theta)I(\theta)^{-1}) \)
   using Frobenius MSE and Bures-squared loss.
2. Required summary plot existence checks.
3. Policy ranking checks vs permutation baseline (`uniform` key).
4. Trajectory file presence and policy completeness checks.

## CLI
- `--config_profile=<profile>`
- `--config_file=<path>`
- `--results_dir=<path>`

## Outputs
- `verification_eq23_summary.csv`
- `verification_required_plots.csv`
- `verification_policy_rank_failures.csv`
- `verification_missing_trajectories.csv`
- `verification_missing_policy_trajectories.csv`
- `verification_report.md`

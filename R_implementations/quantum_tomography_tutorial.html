1. Project Overview

Goal

This project implements adaptive optimal measurement design for quantum state tomography. We compare three measurement selection policies across different loss functions and measurement libraries.

Key Components

Systems

  • One qubit: $N=2$, $d=3$ (Pauli basis)
  • Two qubits: $N=4$, $d=15$ (Pauli products)

Loss Functions

Frobenius (Hilbert-Schmidt)

$$\mathcal{L}_F(\rho, \hat{\rho}) = \|\rho - \hat{\rho}\|_F^2 = \text{tr}\left[(\rho - \hat{\rho})^2\right]$$

Bures (Fidelity-based)

$$\mathcal{L}_B(\rho, \hat{\rho}) = d_B^2(\rho, \hat{\rho}) = 2\left(1 - \text{tr}\sqrt{\sqrt{\rho}\,\hat{\rho}\sqrt{\rho}}\right)$$

Observable-Expectation

$$\mathcal{L}_O(\rho, \hat{\rho}) = \sum_{j=1}^{m} \left(\text{tr}(\rho O_j) - \text{tr}(\hat{\rho} O_j)\right)^2$$

2. Statistical Theory

2.1 Bloch Parametrization

Definition: Bloch Representation

Any $N \times N$ density matrix $\rho$ can be uniquely written as:

$$\rho(\theta) = \frac{1}{N}I_N + \frac{1}{2}\sum_{j=1}^{d} \theta_j \sigma_j$$

where $\{\sigma_j\}_{j=1}^{d}$ are traceless Hermitian matrices satisfying $\text{tr}(\sigma_j \sigma_k) = 2\delta_{jk}$, and $d = N^2 - 1$.

One Qubit ($N=2$, $d=3$)

The Pauli matrices form the basis:

$$X = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}, \quad Y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}, \quad Z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$

The density matrix becomes:

$$\rho(\theta) = \frac{1}{2}\begin{pmatrix} 1 + \theta_3 & \theta_1 - i\theta_2 \\ \theta_1 + i\theta_2 & 1 - \theta_3 \end{pmatrix} = \frac{1}{2}(I + \theta_1 X + \theta_2 Y + \theta_3 Z)$$
Bloch Sphere: The state $\rho(\theta)$ is positive semidefinite if and only if $\|\theta\|_2 \leq 1$. It is strictly positive (full rank) when $\|\theta\|_2 < 1$.

Two Qubits ($N=4$, $d=15$)

The basis consists of 15 Pauli tensor products:

$$\{\sigma_\alpha \otimes \sigma_\beta : (\alpha, \beta) \in \{I, X, Y, Z\}^2 \setminus \{(I,I)\}\}$$

Normalized to satisfy $\text{tr}[(\sigma_\alpha \otimes \sigma_\beta)(\sigma_{\alpha'} \otimes \sigma_{\beta'})] = 4\delta_{\alpha\alpha'}\delta_{\beta\beta'}$:

$$\rho(\theta) = \frac{1}{4}I_4 + \frac{1}{4}\sum_{(\alpha,\beta) \neq (I,I)} \theta_{\alpha\beta}\, \sigma_\alpha \otimes \sigma_\beta$$

2.2 Maximum Likelihood Estimation

Definition: Stabilized MLE with Eigenvalue Floor

Given measurement counts $\{N_{a,b}\}$, the stabilized MLE with floor $\eta > 0$ is:

$$\hat{\rho}_{\text{MLE},\eta} = \underset{\rho \succeq \eta I,\; \text{tr}(\rho)=1}{\arg\min} \left\{ -\sum_{a,b} N_{a,b} \log\text{tr}(\rho\, Q_{a,b}) \right\}$$

where the constraint $\rho \succeq \eta I$ ensures all eigenvalues are at least $\eta$.

Constraint: The eigenvalue floor must satisfy $0 < \eta < \frac{1}{N}$ to ensure feasibility. For $N=2$, use $\eta < 0.5$; for $N=4$, use $\eta < 0.25$.

2.3 Fisher Information

Definition: Per-Shot Fisher Information

The Fisher information matrix for measurement setting $a$ at parameter $\theta$ is:

$$\left[I_a(\theta)\right]_{jk} = \frac{1}{4}\sum_{b=1}^{r_a} \frac{s_j(a,b)\, s_k(a,b)}{p_{a,b}(\theta)}$$

where:

  • $s_j(a,b) = \text{tr}(\sigma_j Q_{a,b})$ is the sensitivity coefficient
  • $p_{a,b}(\theta) = \text{tr}(\rho(\theta) Q_{a,b})$ is the Born probability

In matrix form, using $S_a \in \mathbb{R}^{r_a \times d}$ with $[S_a]_{b,j} = s_j(a,b)$:

$$I_a(\theta) = \frac{1}{4} S_a^\top \text{diag}\left(\frac{1}{p_{a,1}}, \ldots, \frac{1}{p_{a,r_a}}\right) S_a$$

Definition: Total Accumulated Fisher Information

After $n$ measurements with $N_a$ shots at each setting $a$:

$$J_n = \sum_{a=1}^{k} N_a \cdot I_a(\hat{\theta}_n)$$

This is a plug-in estimate using the current MLE $\hat{\theta}_n$.

2.4 Loss Metric Matrices

The asymptotic mean squared error under loss $\mathcal{L}$ is characterized by the metric matrix $G$:

$$\mathbb{E}[\mathcal{L}(\rho^*, \hat{\rho}_n)] \approx \frac{1}{n}\text{tr}(G(\theta^*) \cdot I_\pi(\theta^*)^{-1})$$

Frobenius Metric $G_F$

$$[G_F]_{jk} = \frac{1}{4}\text{tr}(\sigma_j \sigma_k) = \frac{1}{2}\delta_{jk}$$

For an orthonormal basis: $G_F = \frac{1}{2}I_d$

(Constant, independent of $\theta$)

Bures Metric $G_B(\theta)$

$$[G_B(\theta)]_{jk} = \frac{1}{16}\text{tr}\left(\sigma_j \cdot \Omega_{\rho(\theta)}^{-1}(\sigma_k)\right)$$

where $\Omega_\rho(X) = \frac{1}{2}(\rho X + X\rho)$ is the Lyapunov operator.

(State-dependent, must be recomputed)

Observable Metric $G_O$

$$G_O = \frac{1}{4}\sum_{j=1}^{m} o_j o_j^\top, \quad \text{where } [o_j]_k = \text{tr}(\sigma_k O_j)$$

(Constant for fixed observables $\{O_j\}$)

Lyapunov Operator and Its Inverse

The Lyapunov operator maps Hermitian matrices to Hermitian matrices:

$$\Omega_\rho: X \mapsto \frac{1}{2}(\rho X + X\rho)$$

In the eigenbasis of $\rho = \sum_i \lambda_i |i\rangle\langle i|$, the inverse acts as:

$$\left[\Omega_\rho^{-1}(Y)\right]_{ij} = \frac{2}{\lambda_i + \lambda_j} Y_{ij}$$

2.5 A-Optimal Proxy Risk

A-Optimal Proxy Risk

The proxy risk at step $n$ with metric $G$ is:

$$R_n(G) = \text{tr}\left(G(\hat{\theta}_n) \cdot J_n^{-1}\right)$$

This serves as a surrogate for the true expected loss, guiding adaptive measurement selection.

2.6 Adaptive Selection Rules

Policy 1: Uniform Sampling

Select the next measurement setting uniformly at random:

$$a_{n+1} \sim \text{Uniform}\{1, 2, \ldots, k\}$$

(Baseline policy, no adaptation)

Policy 2: Exact One-Step Lookahead

Minimize the proxy risk after adding one measurement:

$$a_{n+1}^{\text{exact}} \in \underset{a \in \{1,\ldots,k\}}{\arg\min}\; \text{tr}\left(G(\hat{\theta}_n) \cdot \left(J_n + I_a(\hat{\theta}_n)\right)^{-1}\right)$$

(Requires $k$ matrix inversions per step)

Policy 3: GI1 (First-Order Approximation)

Using $(J + I_a)^{-1} \approx J^{-1} - J^{-1}I_a J^{-1}$, maximize:

$$a_{n+1}^{\text{GI1}} \in \underset{a \in \{1,\ldots,k\}}{\arg\max}\; \text{tr}\left(G(\hat{\theta}_n) \cdot J_n^{-1} I_a(\hat{\theta}_n) J_n^{-1}\right)$$

(Only requires one matrix inversion per step)

2.7 Initialization Phase

Initialization is deterministic (no uniform or random sampling). We use a fixed, library-specific set of settings so that the accumulated Fisher information $J_n$ is invertible before adaptive selection begins.

4. Simulation Workflow

1
Setup: Build basis $\{\sigma_j\}$ (Pauli/Pauli-product) and measurement library $\{Q_{a,b}\}$
$\downarrow$
2
Generate True State: Sample random full-rank $\rho^*$ from Ginibre ensemble
$\downarrow$
3
Initialization: Deterministic fixed settings per library to make $J_n$ invertible (L1/L2: $X,Y,Z$; L3: all 4 random axes; A: all 15 PVMs; B: all 5 MUB bases)
$\downarrow$
4
Adaptive Loop (for $t = n_0+1$ to $n_{\text{total}}$):
  • Select setting $a_t$ by policy (uniform/exact/GI1) using plug-in $J_t(\hat{\theta}_t)$ and $G(\hat{\theta}_t)$ for exact/GI1
  • Sample outcome $b_t \sim p_{a_t,\cdot}(\rho^*)$
  • Update counts $N_{a,b}$
  • Refit MLE $\hat{\theta}_t$
  • Update plug-in Fisher information $J_t(\hat{\theta}_t)$ and metric $G(\hat{\theta}_t)$ for adaptive scoring
$\downarrow$
5
Monte Carlo (Oracle Evaluation): Repeat for $n_{\text{rep}}$ replicates, compute the mean oracle proxy risk using the true state: $$\bar{R}_n^{\text{oracle}} = \frac{1}{n_{\text{rep}}}\sum_r \text{tr}\!\left(G(\theta^*)\,J_n(\theta^*)^{-1}\right).$$ This oracle risk is what we plot; adaptive selection itself uses the MLE plug-in quantities from Step 4.
$\downarrow$
6
Output: Plot mean oracle risk $\bar{R}_n^{\text{oracle}}(G)$ vs $n$ for three policies

Measurement Library Details

One-Qubit Libraries

L1 (Pauli PVMs): Measure $X$, $Y$, $Z$ observables

$$Q_{u,\pm} = \frac{1}{2}(I \pm \vec{u} \cdot \vec{\sigma}), \quad \vec{u} \in \{\hat{e}_x, \hat{e}_y, \hat{e}_z\}$$

L2 (Nine-axis): $X$, $Y$, $Z$ plus six diagonal directions

$$\vec{u} \in \left\{\hat{e}_x, \hat{e}_y, \hat{e}_z, \frac{\hat{e}_x \pm \hat{e}_y}{\sqrt{2}}, \frac{\hat{e}_x \pm \hat{e}_z}{\sqrt{2}}, \frac{\hat{e}_y \pm \hat{e}_z}{\sqrt{2}}\right\}$$

L3 (Random): Four random unit vectors on Bloch sphere

Two-Qubit Libraries

A (Pauli-parity): All 15 non-identity two-qubit Paulis

$$Q_{P,\pm} = \frac{1}{2}(I_4 \pm P), \quad P \in \{\sigma_\alpha \otimes \sigma_\beta : (\alpha,\beta) \neq (I,I)\}$$

B (5 MUBs): Five mutually unbiased bases for $\mathbb{C}^4$

  • $B_Z$: Computational basis $\{|00\rangle, |01\rangle, |10\rangle, |11\rangle\}$
  • $B_X$: $X \otimes X$ eigenbasis $\{|{+}{+}\rangle, |{+}{-}\rangle, |{-}{+}\rangle, |{-}{-}\rangle\}$
  • $B_Y$: $Y \otimes Y$ eigenbasis
  • $B_{E1}, B_{E2}$: Entangled MUBs

MUB property: $|\langle\psi|\phi\rangle|^2 = \frac{1}{4}$ for vectors from different bases

9. Simulation Setup Details

9.1 Generating Initial Samples

True states are generated using the Ginibre ensemble with identity mixing:

Algorithm: Random Full-Rank Density Matrix

  1. Draw complex Ginibre matrix: $G_{ij} \sim \mathcal{N}(0,1) + i\mathcal{N}(0,1)$
  2. Form Wishart matrix: $W = GG^\dagger$ (positive semidefinite)
  3. Normalize: $\rho_0 = W / \text{tr}(W)$
  4. Mix with identity: $\rho^* = (1-\varepsilon)\rho_0 + \varepsilon \frac{I}{N}$

Default: $\varepsilon = 0.05$

random_density_fullrank <- function(N, eps_mix = 0.05, seed = NULL) {
  # 1. Draw complex Ginibre matrix
  G <- matrix(rnorm(N*N) + 1i*rnorm(N*N), N, N)

  # 2. Form Wishart-type PSD matrix
  W <- G %*% Conj(t(G))

  # 3. Normalize to trace 1
  rho0 <- W / Re(sum(diag(W)))

  # 4. Mix with identity (ensures full rank)
  rho <- (1 - eps_mix) * rho0 + eps_mix * diag(N) / N

  return(rho)
}
Why mix with identity? Mixing with $\varepsilon \cdot I/N$ ensures all eigenvalues are at least $\varepsilon/N$, preventing numerical issues with near-singular states.

9.4 Summary of Mathematical Framework

Key Equations Summary

State Parametrization:

$$\rho(\theta) = \frac{1}{N}I_N + \frac{1}{2}\sum_{j=1}^{d} \theta_j \sigma_j$$

Born Probability:

$$p_{a,b}(\theta) = \text{tr}(\rho(\theta) \cdot Q_{a,b}) = c_{a,b} + \frac{1}{2}\sum_j s_j(a,b) \theta_j$$

Fisher Information:

$$[I_a(\theta)]_{jk} = \frac{1}{4}\sum_b \frac{s_j(a,b) s_k(a,b)}{p_{a,b}(\theta)}, \qquad J_n = \sum_a N_a I_a(\hat{\theta}_n)$$

A-Optimal Proxy Risk:

$$R_n(G) = \text{tr}(G(\hat{\theta}_n) \cdot J_n^{-1})$$

Adaptive Selection (Exact):

$$a_{n+1}^* = \underset{a}{\arg\min}\; \text{tr}\left(G \cdot (J_n + I_a)^{-1}\right)$$

Adaptive Selection (GI1):

$$a_{n+1}^* = \underset{a}{\arg\max}\; \text{tr}\left(G \cdot J_n^{-1} I_a J_n^{-1}\right)$$

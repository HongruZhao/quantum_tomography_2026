Oracle Design Computation (alpha = 0.99)

This page documents the computation logic in compute_oracle_design_alpha99.R that produces the oracle vs uniform comparison table for all 15 cases.

Fixed States

1‑qubit pure state

|+⟩ with Bloch vector (1,0,0)

ρ1q(α) = α |+⟩⟨+| + (1−α) I2/2

2‑qubit pure state

Bell state |Φ⁺⟩ = (|00⟩+|11⟩)/√2

ρ2q(α) = α |Φ⁺⟩⟨Φ⁺| + (1−α) I4/4
Objective

For each (system, library, loss), compute:

R^* = minπ∈Δ tr(G(θ*) [I^π(θ*)]^{-1}),   I^π(θ*) = Σ_a π_a I_a(θ*)
R^{unif} = tr(G(θ*) [I^{unif}(θ*)]^{-1}),   π_{unif} = (1/k,…,1/k)
Ratio = R^{unif} / R^*
Per‑setting Fisher

Using the project’s standard formula:

[I_a(θ)]_{jk} = (1/4) Σ_b s_j(a,b) s_k(a,b) / p_{a,b}(θ)

where p_{a,b}(θ) = tr(ρ(θ) Q_{a,b}) and s_j(a,b) = tr(σ_j Q_{a,b}).

Optimization

The oracle design is solved via a Schur‑complement SDP in CVXR:

minimize tr(G W) s.t. [ I^π I; I W ] ⪰ 0, Σπ=1, π≥0

Implementation uses solve_optimal_design() with the SCS solver.

solve_optimal_design <- function(I_list, G, solver = "SCS") {
  pi <- Variable(k)
  W <- Variable(d, d, symmetric = TRUE)
  I_expr <- Σ pi[i] * I_list[[i]]
  I_block <- bmat(list(list(I_expr, I), list(I, W)))
  minimize tr(G*W) subject to I_block ⪰ 0, Σπ=1, π≥0
}
Workflow
  1. Build library (L1/L2/L3/A/B) and basis (1q or 2q).
  2. Set ρtrue by mixing fixed pure state with maximally mixed (α = 0.99).
  3. Compute θ* = θ(ρtrue), then G(θ*).
  4. Compute per‑setting Fisher I_a(θ*).
  5. Compute uniform risk and oracle risk via SDP.
  6. Store Oracle, Uniform, and Ratio for each case.
Output

The script writes the table to:

R_implementations/results/oracle_design_comparison_alpha_099.csv
This file contains one row per case with Oracle, Uniform, and Ratio.
